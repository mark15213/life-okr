# 人生看板系统设计文档

## 项目概述

一个个人生活追踪看板系统，通过三个核心指标（俯卧撑余额、专注时间、任务完成数）来量化和激励个人成长。系统采用活泼、趣味性强的设计风格，包含"大富翁"式的财富等级系统来增强激励效果。

## 核心指标

### 1. 俯卧撑余额系统
- **规则**：
  - 抽一支烟：+100 俯卧撑
  - 运动一次：-100 俯卧撑
  - 余额为负数时可兑换为等额现金奖励
- **功能**：
  - 显示当前俯卧撑余额
  - 快速操作按钮："+抽烟"、"+运动"
  - 余额为负时显示可兑换金额
  - 显示今日抽烟和运动记录

### 2. 专注时间追踪
- 手动输入每日专注时长（分钟或小时）
- 显示今日累计专注时长
- 显示本周/本月平均专注时长

### 3. 任务完成追踪
- 简单的计数器，记录每日完成任务数
- +1 按钮快速增加任务数
- 显示本周/本月完成任务总数

## 数据模型

### 数据库表结构
```sql
CREATE TABLE daily_records (
  id SERIAL PRIMARY KEY,
  date DATE NOT NULL UNIQUE,
  cigarettes INT DEFAULT 0,        -- 抽烟数
  exercises INT DEFAULT 0,         -- 运动次数
  pushup_balance INT DEFAULT 0,    -- 俯卧撑余额
  focus_minutes INT DEFAULT 0,     -- 专注时长（分钟）
  tasks_completed INT DEFAULT 0,   -- 完成任务数
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

### 数据逻辑
- 每天自动创建当日记录（如果不存在）
- 俯卧撑余额 = 抽烟数 × 100 - 运动次数 × 100
- 所有操作都是增量更新（不是覆盖）

## API设计

### 端点列表
- `GET /api/records?days=7` - 获取最近N天的记录
- `GET /api/records/today` - 获取今日记录
- `POST /api/records/cigarette` - 记录抽烟（+100俯卧撑）
- `POST /api/records/exercise` - 记录运动（-100俯卧撑）
- `POST /api/records/focus` - 记录专注时长
- `POST /api/records/task` - 记录完成任务（+1）

## UI设计

### 视觉风格
- **色彩**：明亮、饱和度高的配色（橙色、青色、紫色、黄色）
- **卡片**：圆角、阴影，轻微3D效果
- **图标**：emoji或插画风格（💪、🚬、⏱️、✅）
- **动画**：数字变化弹跳动画，按钮点击反馈
- **字体**：圆润、友好的字体

### 布局
- **桌面端**：三个卡片横向排列，下方是趋势图
- **移动端**：卡片纵向堆叠

### 趣味元素
- 俯卧撑余额为负时显示庆祝动画
- 连续完成任务时显示"连击"效果
- 数据达到里程碑时显示成就徽章

## "大富翁"财富等级系统

根据俯卧撑余额（负数=可兑换金额），设置不同等级：

| 余额范围 | 等级 | 主题 |
|---------|------|------|
| 0 ~ -500 | 💰 小金库 | 铜色主题 |
| -500 ~ -1000 | 💎 储蓄达人 | 银色主题 |
| -1000 ~ -2000 | 🏆 理财高手 | 金色主题 |
| -2000 ~ -5000 | 👑 财富自由 | 紫金主题 |
| -5000以上 | 🎰 大富翁 | 彩虹特效 |

### 视觉效果
- 每个等级有不同的背景色/渐变
- 升级时触发烟花/金币雨动画
- 卡片边框根据等级发光
- 显示距离下一等级还差多少

## 核心功能详细设计

### 1. 俯卧撑追踪卡片
- 大号显示当前余额
- 余额为负时显示可兑换金额（如：-200 = ¥200）
- 两个大按钮："+抽烟"、"+运动"
- 显示今日统计：今天抽了X支烟，运动了X次
- 显示当前财富等级和进度条

### 2. 专注时间卡片
- 显示今日累计专注时长（HH:MM格式）
- 输入框：输入专注时长（分钟）
- "添加"按钮记录时长
- 显示本周/本月平均专注时长

### 3. 任务完成卡片
- 大号显示今日完成任务数
- "+1"按钮快速增加
- 显示本周/本月完成任务总数
- 连击效果（连续多天完成任务）

### 4. 趋势图表
三条折线图，展示：
- 俯卧撑余额变化（7天/30天可切换）
- 每日专注时长（7天/30天可切换）
- 每日任务完成数（7天/30天可切换）

## 用户体验

### 数据输入
- 所有操作都是手动输入
- 按钮点击即时反馈
- 数据实时更新到云端

### 数据存储
- 云端数据库存储（Vercel Postgres）
- 支持多设备同步
- 无需登录系统（单用户使用）

### 激励机制
- 财富等级系统提供长期目标
- 升级动画提供即时反馈
- 趋势图展示进步轨迹
- 可兑换金额提供实际奖励

## 技术考虑

### 性能
- 服务端渲染（SSR）首屏加载
- 客户端缓存减少API调用
- 图表数据按需加载

### 安全
- API路由验证请求来源
- 数据库连接使用环境变量
- 防止SQL注入（使用参数化查询）

### 可扩展性
- 模块化组件设计
- 易于添加新的追踪指标
- 数据库schema支持扩展
